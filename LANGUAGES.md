# Internationalisation / Translation

FaceChan is currently English-only. Bringing it to other languages is on the roadmap, and community-contributed translations are welcomed now — they will be held ready and merged into the implementation when it lands.

If you would like to contribute a translation, open a pull request adding a language file under `frontend/src/locales/`. No implementation experience needed — if you speak the language fluently and understand the platform, your contribution is valuable.

---
